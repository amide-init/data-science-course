# Lesson 01 — The ndarray (N-dimensional Array)

**Module:** 04 — NumPy
**Duration:** ~40 minutes

---

## Learning Objectives

By the end of this lesson, you will be able to:

- Explain what a NumPy `ndarray` is and how it differs from a Python list
- Create 1D, 2D, and 3D arrays from Python lists
- Inspect an array's core properties: `shape`, `dtype`, `ndim`, `size`
- Explain why NumPy arrays are faster and more memory-efficient than lists
- Reshape an array and understand what reshaping actually changes
