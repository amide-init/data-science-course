# Lesson 04 — Roles in the Industry

**Module:** 01 — Introduction to Data Science  
**Duration:** ~50 minutes

---

## Learning Objectives

By the end of this lesson, you will be able to:

- Name the 6 core data roles and describe what each one does day-to-day
- Distinguish between a Data Analyst, Data Scientist, and ML Engineer
- Explain how a real data team is structured and how roles collaborate
- Identify which role aligns with your own interests and strengths
- Understand what this course prepares you for — and what comes next
