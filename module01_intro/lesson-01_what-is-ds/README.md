# Lesson 01 — What is Data Science?

**Module:** 01 — Introduction to Data Science  
**Duration:** ~45 minutes

---

## Learning Objectives

By the end of this lesson, you will be able to:

- Define Data Science in your own words
- Explain the core steps of a data science workflow
- Give 3 real-world examples of Data Science in action
- Describe what a Data Scientist actually does day-to-day
- Run a basic Python demo that shows data → insight in action
