# Lesson 02 — AI vs ML vs Data Science

**Module:** 01 — Introduction to Data Science  
**Duration:** ~45 minutes

---

## Learning Objectives

By the end of this lesson, you will be able to:

- Define AI, ML, Deep Learning, and Data Science clearly
- Draw and explain the nested relationship between them
- Identify which term applies to a given real-world scenario
- Explain the difference between rule-based AI and learning-based ML
- Know when a project calls for Data Science vs ML vs AI
