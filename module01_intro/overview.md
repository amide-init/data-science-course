# Module 01 — Introduction to Data Science

**Week:** 1  
**Goal:** Students understand what Data Science actually is, where it fits in the industry, and what the full workflow looks like — before writing a single line of analysis code.

## Mini Project

**Name:** Personal Expenses Analysis  
**Folder:** `project/`  
**Description:** Students record their own daily expenses for a week, load them into Python, and answer basic questions: Where does my money go? What's my biggest spending category? Which day do I spend the most?  
**Skills practiced:** Python basics, thinking with data, asking questions from data  
**Dataset:** Student-generated (their own expenses)
