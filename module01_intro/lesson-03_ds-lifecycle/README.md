# Lesson 03 — The Data Science Lifecycle

**Module:** 01 — Introduction to Data Science  
**Duration:** ~50 minutes

---

## Learning Objectives

By the end of this lesson, you will be able to:

- Name and describe all 7 steps of the Data Science lifecycle
- Explain why the lifecycle is a loop, not a straight line
- Identify which step is being performed in a given scenario
- Walk through a complete mini project end-to-end using only Python built-ins
- Recognize where most real projects fail (and why)
