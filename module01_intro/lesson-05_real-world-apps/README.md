# Lesson 05 — Real-World Applications of Data Science

**Module:** 01 — Introduction to Data Science  
**Duration:** ~50 minutes

---

## Learning Objectives

By the end of this lesson, you will be able to:

- Identify data science applications across 10 major industries
- Explain what data is used, what the model does, and who benefits in each case
- Recognise data science working silently in apps and services you use daily
- Build three mini simulations — delivery time estimation, a basic recommender, and a credit scorer
- Think critically about the ethical implications of real-world data applications
