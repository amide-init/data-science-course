# Lesson 06 — Types of Data

**Module:** 01 — Introduction to Data Science  
**Duration:** ~50 minutes

---

## Learning Objectives

By the end of this lesson, you will be able to:

- Draw and explain the full data type taxonomy
- Distinguish structured, semi-structured, and unstructured data
- Classify any column as continuous, discrete, nominal, or ordinal — and explain why
- Explain what goes wrong when you apply the wrong operation to the wrong data type
- Identify Python data types and map them to statistical data types
- Choose the right chart and the right analysis approach for any given column

---

## Files in This Lesson

| File | Description |
|------|-------------|
| `notebook.ipynb` | Main lesson — taxonomy, demos, wrong-type errors, chart guide, exercises |
| `script.py` | Standalone type-checker utility — classify and describe any column |

---

## How to Run

**Option A — Jupyter Notebook (local)**
```bash
jupyter notebook notebook.ipynb
```

**Option B — Run the script directly**
```bash
python script.py
```

**Option C — Google Colab**  
Upload `notebook.ipynb` to [colab.research.google.com](https://colab.research.google.com) and open it.
