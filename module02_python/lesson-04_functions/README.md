# Lesson 04 — Functions

**Module:** 02 — Python for Data Science  
**Duration:** ~60 minutes

---

## Learning Objectives

By the end of this lesson, you will be able to:

- Define and call functions using `def` and `return`
- Use positional, keyword, and default-value parameters
- Return multiple values from a single function
- Explain variable scope (local vs global) and why it matters
- Write clear docstrings so others (and future you) understand your functions
- Use `lambda` for simple one-line functions in sorting and filtering
- Spot and fix the mutable default argument bug — Python's sneakiest gotcha
- Build a reusable data science toolkit: string cleaner, safe converter, stats computer, row validator
