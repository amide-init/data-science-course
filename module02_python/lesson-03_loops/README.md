# Lesson 03 — Loops

**Module:** 02 — Python for Data Science  
**Duration:** ~55 minutes

---

## Learning Objectives

By the end of this lesson, you will be able to:

- Use `for` loops to iterate over lists, strings, and ranges
- Use `range()` with start, stop, and step to control iterations
- Use `enumerate()` to loop with both index and value
- Use `zip()` to loop over two sequences in parallel
- Write `while` loops and avoid infinite loops
- Use `break` and `continue` to control loop flow
- Write list comprehensions as clean alternatives to simple loops
- Apply loops to real data science tasks: computing statistics, counting frequencies, cleaning datasets
