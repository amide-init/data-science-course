# Lesson 02 — Operators and Conditions

**Module:** 02 — Python for Data Science  
**Duration:** ~50 minutes

---

## Learning Objectives

By the end of this lesson, you will be able to:

- Use all arithmetic operators and explain operator precedence
- Write comparison and logical expressions that produce correct boolean results
- Use `in`, `not in`, `is`, `is not` operators confidently
- Write `if`, `elif`, `else` chains to encode business rules
- Chain comparisons the Python way (`0 < score <= 100`)
- Write one-line ternary expressions for simple decisions
- Apply conditions to real data science tasks: data validation, grading, eligibility checks
