# Lesson 01 — Variables and Data Types

**Module:** 02 — Python for Data Science  
**Duration:** ~45 minutes

---

## Learning Objectives

By the end of this lesson, you will be able to:

- Create variables and assign values in Python
- Identify and use Python's five core data types: `int`, `float`, `str`, `bool`, `NoneType`
- Check any value's type using `type()`
- Convert between types using `int()`, `float()`, `str()`, `bool()`
- Explain dynamic typing and why it matters
- Follow Python naming conventions for variables
- Spot and fix the most common type-related errors
