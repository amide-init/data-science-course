# Module 02 — Python for Data Science

**Week:** 2  
**Goal:** Build the complete Python foundation that every future module depends on. By the end of this module, students write clean, working Python programs — not just isolated snippets.

---

## Lessons

| # | Folder | Topic |
|---|--------|-------|

| 2 | `lesson-02_operators-conditions/` | Operators and Conditions |
| 3 | `lesson-03_loops/` | Loops (for, while) |
| 4 | `lesson-04_functions/` | Functions |
| 5 | `lesson-05_lists-tuples-sets/` | Lists, Tuples, Sets |
| 6 | `lesson-06_dictionaries/` | Dictionaries |
| 7 | `lesson-07_exception-handling/` | Exception Handling |

## Projects

| Project | Folder | Description |
|---------|--------|-------------|
| Student Management System | `project-student-mgmt/` | Store, search, and report on student records using all module concepts |
| Expense Tracker | `project-expense-tracker/` | CLI app to log and summarise personal expenses |

Students build **one** of these projects (or both for extra practice).

---

## Estimated Time
- 8 lessons × ~45 min = ~6 hours of class time
- Projects: 2–4 hours outside class

---

## Python Version
- Python 3.8 or above
- No external libraries needed for this module
- All code runs in Jupyter Notebook and as plain `.py` scripts

---

## What This Module Builds Toward
Every concept here is used directly in later modules:
- Variables + types → understand DataFrames (Module 5)
- Loops + conditions → data cleaning logic (Module 6)
- Functions → reusable analysis code throughout the course
- Dictionaries → reading JSON / API responses (Bonus module)
- File handling → loading CSV data before Pandas (Module 5)
