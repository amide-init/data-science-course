# Lesson 01 — Mean, Median, Mode

**Module:** 03 — Mathematics for Data Science  
**Duration:** ~50 minutes

---

## Learning Objectives

By the end of this lesson, you will be able to:

- Define mean, median, and mode precisely and compute each from scratch
- Explain why mean is sensitive to outliers while median is not
- Compute a weighted mean (for GPA calculations, weighted scores)
- Choose the right measure of centre for a given dataset
- Build a `describe()` function that summarises any numeric list
